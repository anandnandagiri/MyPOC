#### Browser Agents
With Browser Agents we can define below
* Browser Version
* OS
* Layout Engine Name & Version
* Hardware Type

#### Browser Layout Engine
* Blink(Chrome, Opera)
* Trident(IE)
* Gecko(Firefox)
* Webkit(Safari)
